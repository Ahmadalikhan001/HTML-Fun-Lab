Tipes for beginnar

Beginner Tip In these examples, you learned these CSS properties: background → Changes the background color. color → Changes the text color. padding → Adds space inside an element. margin → Adds space outside an element. border-radius → Makes rounded corners. box-shadow → Adds a shadow effect. display: flex → Places items in a row. justify-content → Aligns items horizontally. text-align → Aligns text. :hover → Changes the style when the mouse is over an element.
